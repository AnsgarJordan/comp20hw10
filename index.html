API Fun Homework

Purpose

This API uses data from a Russian Government website to inform the user of whether it is currently a holiday in Russia. I found this API on a list of public API's on github. It is part of a work-calendar service.

URL:

https://datazen.katren.ru/calendar/day/
Which gets its data from data.gov.ru

Options

The API returns a JSON object that has information about the current day and whether or not it is a holiday. If the current day is a holiday, the website displays "Yes". If not, it displays "No". Note that the current date is not taken from the users computer, but rather from the website itself.

Uses

This website has a variety of uses. First, it is useful for non-Russians, who may want to know whether or not today is a Russian holiday. This could be important for businesses, who may have to account for an extra business day shipping or receiving goods from Russia if the current day is a holiday. Likewise, since this website is linked to the official Russian data website, citizens of the country will always be up-to-date about holidays in their country.
